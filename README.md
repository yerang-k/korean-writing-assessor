# 바른글 AI — 국어 작문 AI 평가 &amp; 피드백 시스템

2022 개정 교육과정 성취기준에 맞춰 **글쓰기 주제를 추천**받고, 학생이 쓴 글을 **AI가 성취수준(A~E)을 반영해 채점**하며, 그 결과를 **학생별로 구글 스프레드시트에 누적**하는 국어과 수업용 웹앱입니다.

> 🔗 **교사용 화면** — <https://yerang-k.github.io/korean-writing-assessor/?mode=teacher>
> 🔗 **학생용** — 교사가 만든 **공유 링크**로 접속 (별도 설정 없이 바로 사용)

설치가 필요 없는 순수 웹앱이며, 사용법은 앱 안의 **[사용 설명서]** 탭에 단계별로 정리되어 있습니다.

---

## 주요 기능

### 교사
- **2022 성취기준 기반 주제 추천** — 과목과 단원 키워드를 넣으면 성취기준 코드가 붙은 수행평가 주제를 제안
- **AI 평가 기준(루브릭) 자동 생성** — 항목·배점·채점 기준을 만들어 주고, 직접 수정 가능
- **학생용 공유 링크 생성** — 과제·지문·루브릭이 담긴 링크 하나로 배포
- **중복 제출 제어** — 과제별로 '여러 번 제출 허용' 또는 '1회만 제출' 선택
- **과제 보관함** — 설계한 과제를 저장해 두고 나중에 다시 불러오기
- **제출 현황 대시보드** — 과제별·반별·이름으로 조회, 개별 결과 **Word 다운로드**

### 학생
- 학년·반·번호·이름 입력 후 글 작성 (글자 수 실시간 표시)
- 제출하면 즉시 **점수 · 성취수준 · 강점 · 보완할 점 · 문장 첨삭 · 모범 답안** 확인

---

## 시작하기

1. **[설정 &amp; API]** 에서 Gemini API Key 등록
   *(무료 발급: <https://aistudio.google.com/apikey>)*
2. **[교사 모드]** 에서 주제·지문·논제를 입력하고 평가 기준 생성
3. **학생용 공유 링크**를 만들어 배포 → 학생이 제출하면 자동 채점

자세한 절차와 문제 해결은 앱의 **[사용 설명서]** 탭을 참고하세요.

---

## 구글 스프레드시트 연동 (선택)

학생 제출 기록을 계속 누적하려면 스프레드시트를 연동합니다.

- 앱 **[설정 &amp; API] → 구글 스프레드시트 연동 설정** 안내에 있는 **Apps Script 코드**를 복사해 배포하면 됩니다.
- 제출 시 **과제명으로 된 탭이 자동 생성**되어 과제별로 기록이 분리됩니다.
- 시트에는 제출시간·학년·반·번호·이름·과제명·글자수·총점·성취수준·항목별 점수·강점·개선점·첨삭·모범답안·작성글이 저장됩니다.

> 배포 시 **액세스 권한을 '모든 사용자'** 로 설정하고, 주소가 `/exec` 로 끝나는지 확인하세요.

## AI 프록시 (권장 · 선택)

공유 링크에 API Key를 담으면 키가 노출될 수 있습니다. **Cloudflare Worker 프록시**를 연결하면 키가 서버에만 저장되어 **학생에게 전달되지 않습니다.**

- 앱 **[설정 &amp; API] → AI 프록시 연결** 에 Worker 코드와 배포 방법이 들어 있습니다.
- 프록시를 쓰면 학생 링크에는 **프록시 주소와 접근 토큰**만 담기며, 토큰이 유출돼도 **토큰만 교체**하면 됩니다.

> ⚠️ 프록시용 키는 Google Cloud에서 **애플리케이션 제한사항을 '없음'** 으로 두어야 합니다. (서버에서 호출하므로 HTTP 리퍼러 제한을 걸면 차단됩니다.)

## 성취수준 판정 기준

AI가 매긴 총점의 **성취율(획득 점수 ÷ 총 배점)** 로 자동 산출되며, 항목별로도 함께 표시됩니다.

| 성취수준 | A | B | C | D | E |
|---|---|---|---|---|---|
| 기준 | 90% 이상 | 80% 이상 | 70% 이상 | 60% 이상 | 60% 미만 |
| 의미 | 매우 우수 | 우수 | 보통 | 기초 | 노력 요함 |

## 저장소 구조

```
index.html      실제 앱 화면 (GitHub Pages가 서비스하는 파일)
app.js          앱 로직 (AI 호출·채점·시트 연동·화면 제어)

```

## 기술 구성

- **프런트엔드** — 순수 HTML / CSS / JavaScript (빌드 도구 없음), GitHub Pages 배포
- **AI** — Google Gemini API (사용 가능한 모델 자동 탐색)
- **데이터 저장** — Google Apps Script 웹앱 + 구글 스프레드시트
- **보안(선택)** — Cloudflare Workers 프록시로 API Key 은닉
