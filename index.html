구글 스프레드시트 연동 설정 (선택사항)

학생들의 작문 제출 내용과 AI 채점 결과를 교사의 구글 스프레드시트에 실시간으로 누적하고, 교사 모드의 '제출 현황' 탭에서 모아볼 수 있습니다.

🛠️ 구글 스프레드시트 연동 방법 보기 (클릭)
  1. 구글 스프레드시트를 생성하고, 첫 번째 행(1행)에 아래 항목을 순서대로 적습니다:
    제출시간, 학년, 반, 번호, 이름, 과제명, 글자수, 총점, 성취수준, 평가기준별점수, 강점, 개선점, 첨삭내용, 모범답안, 작성글
  2. 상단 메뉴에서 [확장 프로그램] → [Apps Script]를 클릭합니다.
  3. 기존 코드를 모두 지우고 아래의 Apps Script 코드를 복사해서 붙여넣습니다.
  1. 우측 상단의 [배포] → [새 배포]를 클릭합니다.
  2. 유형 선택(톱니바퀴)에서 [웹 앱]을 선택합니다.
  3. 액세스할 수 있는 사용자를 반드시 [모든 사용자(Anyone)]로 설정한 후 배포합니다.
  4. 권한 승인 과정을 거치면 발급되는 웹 앱 URL을 복사하여 위의 입력창에 붙여넣고 저장해 주세요!
